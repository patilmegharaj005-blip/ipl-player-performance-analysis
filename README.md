🚀 Project Overview

This project focuses on performing Exploratory Data Analysis (EDA) on Indian Premier League (IPL) player data to extract meaningful insights into batting performance and match trends.

Using powerful Python libraries, raw cricket data is transformed into clear, visual, and actionable insights that help understand player consistency, scoring behavior, and season-wise performance.

🎯 Objectives
Analyze player performance using key batting metrics
Identify top-performing players across IPL seasons
Study scoring patterns and trends
Explore relationships between performance indicators
Present insights using interactive visualizations

🛠️ Tech Stack
Python
Pandas (Data Manipulation)
Matplotlib (Data Visualization)
Seaborn (Statistical Visualization)
Jupyter Notebook / Google Colab

📊 Key Highlights
✔ Data Cleaning & Preprocessing
✔ Exploratory Data Analysis (EDA)
✔ Feature Understanding & Transformation
✔ Data Visualization using multiple chart types
✔ Player Performance Evaluation
✔ Season-wise Trend Analysis

📈 Key Insights
📌 Players with higher strike rates and boundary counts contribute significantly to total runs
📌 A small group of players shows consistent high performance across seasons
📌 Strong correlation between runs, strike rate, and boundaries
📌 Seasonal trends indicate evolving gameplay dynamics in IPL

📉 Visualizations Included
📊 Histogram (Runs Distribution)
📊 Bar Chart (Top Players by Runs & Centuries)
📊 Scatter Plot (Strike Rate vs Runs, Average vs Runs)
📊 Line Chart (Season-wise Total Runs)
📊 Heatmap (Correlation Analysis)
